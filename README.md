# TCG Companion

Multi-game tournament tracker for TCG players.

## Quick Start

```bash
npm install
npm run dev
# Visit http://localhost:8787
```

## Features

✅ Multi-game support (Pokemon, Magic, Riftbound)
✅ Meta tracking with Limitless API
✅ Firebase Auth + Firestore
✅ Tournament & Match tracking
✅ Win rate stats

## Setup Firebase (Optional)

See `FIREBASE_SETUP.md` for full instructions.

## Deploy

```bash
npm run deploy
```
